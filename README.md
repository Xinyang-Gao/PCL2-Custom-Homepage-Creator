# TODO List

## 🟡 健壮性与性能（P2）

### 5. XAML 解析错误行列定位
- **文件**：`script.js` → `XamlProcessor.importFromXAML`
- **问题**：错误信息仅截取前 100 字符，无行列号，用户难以定位。
- **计划**：从 `parsererror` 元素中提取 `lineNumber` / `columnNumber`（部分浏览器支持），若不可用则尝试解析错误文本中的行/列信息。
- **预计工时**：1.5h

### 6. 后端并发写入原子性
- **文件**：`app.py` → `/api/backup`、`/api/save`、`/api/local/save`
- **问题**：多请求同时写同一文件可能交错损坏内容。
- **计划**：写入临时文件（`filepath.tmp`）后使用 `os.replace` 原子替换，或引入 `portalocker`。
- **预计工时**：1h

### 7. 属性面板异步渲染（防卡顿）
- **文件**：`script.js` → `RenderManager.updatePropsPanel`
- **问题**：复杂组件树（>50 组件）切换时同步构建 DOM 导致短暂空白。
- **计划**：使用 `requestAnimationFrame` 分批渲染属性分组，或对长列表虚拟滚动。
- **预计工时**：1.5h

### 8. 组件复制 ID 动态基准（可选）
- **文件**：`script.js` → `ComponentManager.deepCloneComponent`
- **问题**：当前 `nextId` 仅在初始化和导入时更新，连续复制不会冲突，但若外部导入 ID 跳跃较大且未触发 `nextId` 更新，理论上可能重复。
- **计划**：在 `deepCloneComponent` 开头调用 `getMaxGlobalId() + 1` 确保基准最新。
- **预计工时**：0.5h
